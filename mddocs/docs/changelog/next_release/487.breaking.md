Replace `SFTP(timeout=..., host_key_check=..., compress=...)` with new class:
```python
SFTP(
  extra=SFTP.Extra(
    timeout=...,
    host_key_check=...,
    compress=...,
    banner_timeout=...,
    auth_timeout=...,
    channel_timeout=...,
  ),
)
```

This new class allows passing additional params directly to `paramiko.SSHClient`.
